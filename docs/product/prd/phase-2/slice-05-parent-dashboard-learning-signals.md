# Slice 5: Parent Dashboard & Learning Signals

## Overview
Parent-facing dashboard showing SR progress, weak items, and reading signals.

## Contents
- Parent can unlock dashboard (behind a simple gate).
- Dashboard shows:
  - **Overview:** Sessions played this week, average session length.
  - **Letter-Sound Knowledge:** "Strong: m, a, t, s. Growing: p, i. Needs practice: n, o."
  - **Weak items:** "Often misses: /o/, /i/. Try again tomorrow."
  - **Suggested action:** "Practice /i/ with these sounds. Point them out around the house."
- All data is pulled directly from SR engine (no manual work).
- Parent can optionally trigger a "focus quest" on a specific weak letter.
